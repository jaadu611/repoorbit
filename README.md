# 🛰️ RepoOrbit

**Visualize the DNA of your codebase.**

RepoOrbit is an open-source, physics-based repository explorer that transforms flat GitHub file structures into interactive, multidimensional maps. Built for developers who need to audit architecture, identify technical debt, or understand project evolution at a glance.

## 🚀 Key Features

- **Chrono-Tree:** A temporal visualization showing the growth of folders and files over time.
- **Trace Mode:** A dependency mapper that visualizes imports and exports with directed arrows.

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Visualization:** D3.js & react-force-graph
- **API:** GitHub REST API via Octokit
- **Styling:** Tailwind CSS

## 🏁 Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```
